# Schedule comparison and what-if analysis

Tools for comparing schedules and analyzing alternatives.

## Features
- [ ] Schedule diff: compare two schedules
- [ ] What-if analysis: impact of constraint changes
- [ ] Sensitivity analysis: which constraints are tight
- [ ] Alternative solutions viewer

## Use Cases
- Compare before/after manual adjustments
- Evaluate impact of adding/removing workers
- Understand constraint trade-offs
